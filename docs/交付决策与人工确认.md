## 2. 需要人工确认或补充

### 2.1 港湾真实数据

当前仓库内 HB001-HB008 仍为演示样例数据。后续要变成更可信的作品集，需要根据地图搜索“劳动者港湾”确认：

- 港湾名称
- 区县
- 商圈或街道
- 详细地址
- 经纬度
- 开放时间
- 可用设施
- 数据来源截图或链接

建议先覆盖重庆主城九区，每区至少 2 个点，共 18 个左右。

### 2.2 地图服务

你需要后续任选一个服务商并申请 Web/H5 Key：

- 高德开放平台
- 腾讯位置服务
- 百度地图开放平台

拿到 Key 后填入 `.env.local`，再把当前 mock 导航替换为真实路线规划。

### 2.3 天气服务

天气不是首版阻塞项。后续如果需要真实天气，可申请：

- 和风天气
- 高德天气
- OpenWeather

拿到 Key 后填入 `.env.local`，再把当前文本触发天气换为实时天气。

### 2.4 GitHub Pages

仓库已加入自动部署配置。你需要在 GitHub 仓库设置中确认：

- Settings -> Pages
- Source 选择 GitHub Actions
- 推送 main 分支后等待 Actions 完成
